Usage Example

// Load the icon mapping system const icon = IconMapping.getIconForPioneer({ name: "Marie Curie", fields: ["Physics", "Chemistry"], region: "Poland", birthYear: 1867 }); // Result: "👩🏽‍🔬" (Single emoji character with medium skin tone) console.log(icon);
// Get random diverse icon for variety const randomIcon = IconMapping.getRandomDiverseIcon(); // Result: Random single emoji character with appropriate skin tone console.log(randomIcon);

Field Mapping Logic

The system maps pioneer fields to appropriate icons:

  • Biology/Chemistry/Physics → 👩‍🔬 Scientist
  • Computer Science/Programming → 👩‍💻 Technologist
  • Engineering → 👩‍🏭 Engineer
  • Astronomy/Space → 👩‍🚀 Astronaut
  • Medicine/Healthcare → 👩‍⚕️ Doctor
  • Education/Academia → 👩‍🏫 Professor
  • Mathematics → 👩‍🎓 Mathematician
  • Invention/Innovation → 👩‍🔧 Inventor
